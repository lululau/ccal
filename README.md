# ccal

一个查看中国农历的命令行日历工具

## 用法

    显示当前月的日历:   
       ccal
    显示当前年的日历:
       ccal -y
    显示当前年9月份的日历:
       ccal 9
    显示公元1983年的日历:
       ccal 1983
    显示公元2012年12月份的日历:
       ccal 2012 12
    显示公元9年的日历:
       ccal -y 9

## 安装

### macOS

```
brew install lululau/utils/ccal
```

### Linux

```
git clone https://github.com/lululau/ccal
cd ccal/
sudo cp ccal /usr/local/bin
cd lib/
perl Makefile.PL
make
sudo make install

ccal --help
```
